## React + Vite + Tailwind Portfolio

This project is a modern single-page portfolio application built with React and powered by Vite for fast development and optimized production builds. It follows a component-driven architecture and uses Tailwind CSS for utility-first styling to maintain consistency and scalability across the UI.

The application separates reusable components, page-level sections, and static data into clearly defined directories. Project and experience data are centralized to simplify updates without modifying presentation logic, improving maintainability and long-term extensibility.

Animations and visual effects are implemented using lightweight motion libraries and custom components, providing smooth interactions while preserving performance. The overall structure is designed to reflect real-world frontend engineering practices rather than a simple static site.
